# ADR 0015 — Code of conduct: deferred

- **Status:** Accepted
- **Date:** 2026-04-20

## Context

Most mature OSS projects publish a code of conduct (often Contributor Covenant 2.1). At pre-alpha with effectively one contributor, a formal CoC adds process without a community to apply it to.

## Decision

No CoC file at this stage. Revisit when the contributor base grows meaningfully (5+ regular contributors, or any community incident).

## Consequences

- Lower friction to start. The `CONTRIBUTING.md` ends with "be decent to each other" as informal guidance.
- When the project grows, the adoption of a CoC will itself be an ADR — that ADR will specify the text (likely Contributor Covenant 2.1), enforcement, and contacts.

## Alternatives considered

- **Adopt Contributor Covenant 2.1 now.** Harmless but performative with no contributors to govern.
